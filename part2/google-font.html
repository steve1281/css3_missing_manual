Google Font

Using Google fonts is pretty (pretty) straight forward (forward).

Start on the page http://www.google.com/fonts/#

Then select the font, say use, choose the specifics. You should choose all the fonts you need. Google will produce a link and some examples.

You can also download the google fonts, and use a process similiar to Custom Font Example to use it.
